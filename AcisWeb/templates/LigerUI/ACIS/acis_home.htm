ACIS

ACIS full name Automatic Continuous Integration System, an automated test system that focuses on testing around the SWI-Module. The ACIS web is part of the ACIS system and plays the role of data display, node monitoring and problem tracking etc...

  • One developer, mulitple projects

  • ACIS maintainer monitor ACIS status
